# 发文百科

面对AO3复杂的发布界面，望而却步者有之，怨声载道者有之，赞不绝口者有之。无论如何，如此体量的创作平台，手续多是一种必然。这一节的目的就是厘清这「一团乱麻」，帮助任何有意加入AO3社群的创作者实现无障碍发文。

如果准备好了，就点开神圣的`Post`按钮吧！

![](../../.gitbook/assets/Screenshot\_20230313\_195516.jpg)

然后点击`New Work`，来到发布新作品的页面。
