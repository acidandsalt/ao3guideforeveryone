# 遗弃作品（Orphan Work）

{% hint style="danger" %}
<mark style="color:red;">**注意：这个功能一旦使用就无法撤销，请慎重考虑！**</mark>
{% endhint %}

## 功能

在使用遗弃作品功能后，作品将会从你的账号名下移除，转移到专门的orphan\_account之下。遗弃作品意味着你放弃对这个作品的一切关联和权力，你将：

* 无法再编辑或删除这篇作品
* 不再被显示为这篇作品的创作者
* **无法撤销这个选择**

## 做法

1. 进入作品编辑界面，点击`Orphan Work`。

<figure><img src="../../.gitbook/assets/MTXX_MH20230315_144010958.jpg" alt="" width="375"><figcaption></figcaption></figure>

2. 选择是否要保留用户名，然后点击`Yes, I'm sure`。
